\section{Travail}

\textbf{Exercice 1.} Changer de manière récursive les droits des fichiers de
type texte (extension .txt) sur une arboresence. On indiquera au script le
répertoire d'où doit commencer le changement.\\

Voici un exemple d'appel :
\begin{lstlisting}[language=bash]
$ ./change_permissions.pl mon_repertoire
\end{lstlisting}
